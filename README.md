# Puppeteer Studying
Repositório de estudos em Puppeteer.

## Tecnologias utilizadas:

#### Dependências:

- puppeteer-extra
- puppeteer-extra-plugin-adblocker

#### Dependências de desenvolvimento:

- nodemon

## Como rodar a aplicação:

- Antes, lembre-se de instalar os módulos Node do projeto:

```bash
 npm install
```

- Rode o comando:

```bash
 npm run dev
```
ou
```bash
 node index.js
```
